MCP tools
ctx_tree_read
Read file with tree-sitter semantic chunking. Returns nodeId + symbol index; chunks stored for future compose() calls.
ctx_tree_grep
Ripgrep search with results stored as graph nodes. Each matching file links to its semantic chunk stream.
ctx_tree_browse
Fetch a URL and store it as a web_content node with accessibility tree representation. Returns compact reference.
ctx_tree_search
FTS5 keyword search across all stored nodes — current session and all prior sessions for this project.
ctx_tree_compose
Budget-bounded context bundle from seed nodeIds. Assembles a focused context slice without re-reading anything.
ctx_tree_neighbors
Graph walk from a node — find callers, imports, same-module siblings, derived nodes, and summarized subgraphs.
ctx_tree_path_to_root
Walk the parent chain from any node to the session root. Understand how a chunk fits into the larger picture.
ctx_tree_recent
Surface nodes captured in prior sessions. Instant orientation without re-reading anything from last time.
ctx_tree_monitor
Run a command and stream output as chunk nodes. Read-write mode supports stdin sends in the same graph.
Hooks — fires automatically, zero configuration after install
SessionStart
On every session startup or resume
Injects the ctx-tree tool guide and redirect reminders into context.
UserPromptSubmit
User submits a prompt ≥ 100 chars
Stores prompt as observation node. Injects nodeId so subagents reference it without re-reading. Surfaces in future sessions via ctx_tree_search.
PreToolUse
Read (code files) · Grep · Bash grep/cat · WebFetch · Monitor
Hard-denies native tool. Emits exact ctx_tree_* replacement call as additionalContext — Claude retries immediately with the correct tool.
PreToolUse
Agent tool
Injects ctx-tree tool map + FTS context hits from prior work into subagent prompt. Creates session node. Writes state file for capture hook.
PreToolUse
Skill tool
Cache hit: denies and injects ctx_tree_compose reference — zero disk I/O. Cache miss: passthrough, first load stores the skill for next time.
PostToolUse
Agent tool completes
Creates summary note node linking all subagent work. ctx_tree_neighbors(sessionId) surfaces the full subagent graph from the parent session.
PostToolUse
Skill tool completes · WebSearch completes
Skill: stores content as cached node for instant recall next time. WebSearch: fire-and-forget browse of each result URL — pages available in future searches.
